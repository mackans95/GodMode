# My Time at Sandrock - GodMode Mod

Infinite health and stamina (basically) with no extra setup!

## Installation
1. Make sure you have BepInEx installed in your game folder (Where Sandrock.exe is located)
1. Copy `GodMode.dll` to `BepInEx/plugins/` in your game folder.
2. Launch the game. Make sure the mod is enabled in the config if it's not working

## Configuration
- Enabled: true/false
- IsDebug: true/false (shows logs)
