# \# GodMode Mod

# Instant God Mode for \[Sandrock].

# 

# \## Installation

# 1\. Copy `GodMode.dll` to `BepInEx/plugins/` in your game folder.

# 2\. Launch the game. Make sure the mod is enabled in the config.

# 

# \## Configuration

# \- Enabled: true/false

# \- IsDebug: true/false (shows logs)
